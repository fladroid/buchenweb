X-Ray — Score Guide
BT
Back Translation — the winning translation translated back to English. Shows how much meaning survives the round trip.
Translation Score
Cosine similarity between the translation and the original in the multilingual embedding space (e5-large). Measures semantic closeness across languages.
Back Score
Cosine similarity between the back translation and the original English. A high score means the meaning survived the round trip intact.
Judge Average
Average of the three judge scores below, assigned by the LLM judge (gemma4:31b). The primary quality signal from the judge.
Grammar
Judge score for grammatical correctness and syntactic quality of the translation (0–1).
Naturalness
Judge score for how natural and fluent the translation reads to a native speaker of the target language (0–1).
Fidelity
Judge score for how faithfully the translation conveys the full meaning of the original sentence (0–1).
Model
The winning translation model: gemma3 (Google), ministral (Mistral AI), or nllb-600M (Meta NLLB — local CPU model). Variants marked -refine are self-refine candidates: the same model re-translating with the current winner supplied as a reference (see Self-Refine).
Self-Refine
An experiment in anchored mutation: a model re-translates a sentence with the existing winner given as a hint, trying to improve on it while staying grammatically valid. On strong winners it rarely wins (the anchor is already near-ceiling), but it shows that guided variation over language stays inside the grammatical space — the missing piece for evolutionary approaches to translation. A genuine X-Ray exhibit: shown, scored, and honest about when it does not beat the original.
t=
Temperature at which the model was run. Lower values (0.1) produce more deterministic output; higher values (0.8) introduce more variation and creativity.
